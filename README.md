# vda5050-lif-zod

Zod schemas and TypeScript types for VDA5050 and LIF (Layout Interchange Format).

## Usage

Go to the `src` folder and copy the schema file you need into your project.

- [`src/vda5050.ts`](./src/vda5050.ts) — VDA5050 schemas (**TODO: not yet implemented**)
- [`src/lif.ts`](./src/lif.ts) — LIF schemas

## References

The official guideline for LIF can be found in the VDMA publication, which is available free of charge: [LIF Publication](https://www.vdma.eu/documents/34570/3317035/FuI_Guideline_LIF_GB.pdf).

The original VDA 5050 specification is published and maintained in the official repository by VDA/VDMA: [VDA 5050 Specification](https://github.com/VDA5050/VDA5050).
